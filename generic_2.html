我的網頁前端設計

這是我第一次開發網頁，因為才剛入門所以是使用模板改良，套用之前在社團學到的前端設計能力加以修改成我自己的專屬網站，並且在學長的教學下使用 Github 提供的免費網域進行開發。

Step1. 在Github開一個新的repository， 然後命名repository

Step2.用預設模板來建立靜態網頁接下來我們回到github的網頁，切換到Settings頁面往下滑到GitHub Pages的區塊

按下圖片左下角的“Choose a theme”讓github先建立一個由幾種模板產生的靜態網站接著會跳到下面這個頁面，選擇Github Pages的主題這邊隨便選一個就可以了預設其實就OK，反正之後會變成自己的網頁然後按下右邊的“Select theme”按鈕

這時候用 https://{我的帳號名稱}.github.io/{我的repository名稱}/ 來存取你剛剛選擇的模板所建立的網站

Step3.綁定申請好的網域 接著回到repository的Settings頁面 可以看到Gihub Pages多了一些設定欄位 在custom domain的欄位中，填入domain Name 然後按下右邊的”Save”

Step4.複製repository到電腦中 這個步驟開始邊會使用到Github for Desktop

接著電腦中的Github Desktop會被喚醒 出現類似以下的畫面

基本上都使用預設就可以了 Local Path的部分就是repository下載下來之後儲存的位置 可以自己調整 最後再按下右下角的“Clone”按鈕，開始複製一份repository到電腦後就可以開始開發了

Step5.部署靜態網站 一切都準備就緒了， 剩下只差把我們的網站放到Github上代管了！ 接下來開啟Step3複製成功的repository資料夾， 把網站的首頁命名成index.html， 接著開啟Github Desktop， 會出現類似以下的畫面

然後按照1(紀錄的標題),2(紀錄的內容),3(儲存記錄)的順序進行填寫或是點擊 3的部分儲存完成之後，4的介面會變成這樣

Push origin後， loading完成就代表成功將檔案同步至Github， 可以開啟或是重新整理一下網站， 應該就可以看到你做好的網站

Step6.用vscode打開本地檔案進行開發，當要更新版本時直接push到Github就會完成更新
